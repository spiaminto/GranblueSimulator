support ability

적의 서포트 어빌리티는 고유버프 및 기믹을 구현하기 위한 Move 이기 때문에 필요한 부분만 작성하면됨

현재 구현목적 : 배틀 시작시 스테이터스 부여 (스테이터스가 Move 에 종속되있어서 어쩔수 없이 Move 에 저장하고 발동시켜야됨)

적 id
type: 어빌리티, 보스는 서포트어빌리티만 Move.type (FIRST_SUPPORT_ABILITY, ...)
name: 어빌리티 이름 -> 현재 사용하지 않음
info: 어빌리티 설명 텍스트 -> 현재 사용하지 않음
damageRate: 데미지 배율
hitCount: 히트수
coolDown: 어빌리티 쿨타임
duration: 어빌리티 효과시간 (두개 이상일 경우 쉼표로 구분하여 작성, 로직용 아니고 표현용)
iconSrc: asset.iconImageSrc 에서 사용
이펙트 비디오 asset.effectVideo
모션 비디오 asset.motionVideo
SE 오디오 asset.seAudioSrc
Voice 오디오 asset.voiceAudioSrc
